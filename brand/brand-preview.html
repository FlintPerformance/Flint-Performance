06
PRODUCTION NOTES

Before exporting to PNG/JPEG for upload:

  • Open SVG in Figma (File → Import) or Adobe Illustrator (File → Open)
  • Ensure Google Fonts are loaded: Barlow Condensed 900 and DM Sans
  • Select all text layers → Type → Create Outlines (Illustrator) or Flatten (Figma) before final export
  • Export at 2× minimum for retina screens — or use SVG directly where supported
  • PNG is required for LinkedIn, Facebook, X profile pictures and covers (they don't accept SVG)
  • YouTube channel art: export as PNG, max file size 6MB per YouTube spec
  • The period . is always #f04a0e — never change this
  • Minimum logo width: 120px in any application
